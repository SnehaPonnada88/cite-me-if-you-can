### `POST /api/upload`

Upload a JSON file of text chunks to populate the Qdrant collection.  
Sample format:

```json
[
  {
    "text": "Velvet bean helps fix nitrogen...",
    "source_doc_id": "AGRO001",
    "section_heading": "Soil Benefits",
    "journal": "AgroScience Journal",
    "publish_year": 2020
  },
  ...
]
